Инновационная экскурсия в мир робототехники

Проект “GAR-5. Экскурсия по робостанции” от Московского Политеха - это интерактивная экскурсия по павильону “Робостанция”, который находится на ВДНХ. С помощью приложения, разработанного студентами из проекта GAR-5, посетитель сможет с комфортом посетить павильон.

О проекте

Проект GAR-5 "Экскурсия по Робостанции" - это мобильное приложение-гид для посетителей младшего школьного возраста павильона "Робостанция" на ВДНХ. GAR-5 предлагает получить им полезную информацию о роботах с помощью робота-экскурсовода в своем телефоне.

С помощью AR-анимации и адаптированных аудиорассказов дети младшего школьного возраста в доступной форме узнают о технологиях, взаимодействуют с экспонатами и делают запоминающиеся фото с виртуальными роботами. Наша система позволяет каждому ребенку получить персонализированную экскурсию, соответствующую его уровню понимания и интересам.

Проект разрабатывается студентами и сочетает образовательную функцию с цифровыми развлечениями, делая посещение выставки более современным и вовлекающим. Мы стремимся сделать изучение робототехники увлекательным и доступным для всех возрастов.

Участники проекта

Над проектом GAR-5 работает междисциплинарная команда специалистов, объединяющая экспертов в различных областях технологий и дизайна:

Программисты

Разработка приложения и настройка робота в движке Unity. Создание алгоритмов навигации, обработки голосовых команд и интеграции с AR-технологиями. Оптимизация производительности системы.

3D-дизайнеры

Создание 3D модели робота-экскурсовода и анимаций для него. Разработка визуальных эффектов для дополненной реальности. Моделирование интерактивных элементов экспозиции.

2D-дизайнеры

Создание дизайна рамок для QR-кодов и внешнее оформление. Разработка пользовательского интерфейса приложения. Визуальная идентификация проекта и создание маркетинговых материалов.

Журнал проекта

Студенты Московского Политеха разработали 3D-модель "Робота-экскурсовода"

Идея создания виртуального гида возникла с целью сделать музейные экспозиции более привлекательными и понятными для детской аудитории.

«Робот-экскурсовод» представляет собой мобильное приложение с 3D-моделью дружелюбного робота, который знакомится с юными посетителями и в интерактивной форме рассказывает им об экспонатах. Пользователи могут в произвольном порядке изучать заинтересовавшие их объекты, сканируя QR-коды и получая информацию в текстовом, аудио и видео форматах.

Меняется вектор направления проекта с "Разработки приложения" на "Поддержку приложения"

Разработка мобильного приложения успешно завершена и теперь команда проекта сосредоточится на поддержке и развитии проекта: улучшении функционала, оптимизации работы системы и внедрении новых возможностей.

Завершена разработка приложения "Робот-экскурсовод"

Завершена разработка приложения "Робот-экскурсовод" для павильона "Робостанция" на ВДНХ. Новое приложение позволит посетителям получать интерактивные экскурсии и пользоваться AR-подсказками.
